# Customer Churn Prediction

## Problem Statement

Predict whether a customer will leave the company or not

---

## Dataset Features

* Age
* Tenure
* Balance
* Products
* Active Status

---

## Technologies

* Python
* Pandas
* Scikit-learn

---

## Workflow

1. Data Loading
2. Data Preprocessing
3. Model Training (Logistic Regression)
4. Model Evaluation

---

## Results

* Accuracy: ~80%
* Confusion Matrix used for evaluation

---

## Conclusion

* Customers with low activity are more likely to churn
